Данное консольное приложение выводит характеристики фигуры считываемой из файла.
На вход подаётся тип вывода информации (консоль -с или файл -f), и входный файл в котором должны быть 2 строки:
1 строка - тип фигуры
2 строка - параметры фигуры
Если тип вывода не указан то информация выведетсяв консоль.

Если указан тип вывода информации - файл, он будет создан и назван output.txt
Пример содержимого файла:
CIRCLE
5

Общие характеристики для всех типов фигур:
•Название
•Площадь
•Периметр

Поддерживаемые типы и параметры для фигур:

Тип - CIRCLE
Параметры - Радиус
Выходная информация
- Диаметр

Тип - RECTANGLE
Параметры - Длины сторон (два значения)
Выходная информация:
- Длина диагонали
- Длина (размер длинной стороны)
- Ширина (размер короткой стороны)

Тип - TRIANGLE
Параметры - Длины сторон (три значения)
Выходная информация для каждой из трех сторон:
- Длина стороны
- противолежащий угол
